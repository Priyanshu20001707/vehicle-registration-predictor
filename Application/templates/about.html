About This Project

This project is built using a large dataset of Telangana vehicle registrations from 2021, containing approximately 1.4 million records. The data was preprocessed, analyzed, and used to train several machine learning models to predict:

  • Vehicle registration count
  • Usage type (Private or Commercial)
  • Registration duration
  • Fuel group

The app is built with Python, Flask, and scikit-learn, with a modern web UI using HTML, CSS, and responsive design techniques.

The goal is to provide quick and useful insights for future vehicle registration trends and assist stakeholders with data-driven decisions.
